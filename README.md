# Akalenchuk — Subtitles

English subtitles for Japanese variety shows.

These subtitles were generated as follows:
- Japanese text transcribed from audio via Whisper V3. Tool: MacWhisper (free)
- Translation by the (full) latest GPT model (currently 5.4). Tool: [gptsubtitler.com](https://gptsubtitler.com) (free, but the GPT API is somewhat expensive at ~$0.60 per hour of video)
- Subtitles manually retimed, split, gaps filled, translations corrected by myself and my Japanese wife. A lot of work spent here, but as the above two steps continue to get better this will become easier. 

Note, I'll be giving the [ChigyuSubs-pipeline](https://github.com/chigyusubs/chigyusubs-pipeline) a try for the next round of shows -- if I can figure out how to run it! I'm a bit of a dummy. :)
