# Akalenchuk — Subtitles

English subtitles for Japanese variety shows.

Inspired by ChigyuSubs work, these subtitles were generated as follows:
- Japanese text transcribed from audio via Whisper V3
- Translation by the (full) latest GPT model (currently 5.4). Somewhat expensive at around $0.60 per hour :(
- Subtitles manually retimed, split, gaps filled, translations corrected by myself and my Japanese wife. A lot of work spent here, but as the above two steps continue to get better this will become easier.
